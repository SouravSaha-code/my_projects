🔬 Overview

This application uses advanced deep learning technology to classify cervical cancer cells from medical images. The system employs an ensemble of state-of-the-art convolutional neural networks to provide accurate and reliable predictions.

📊 Model Performance

94.23% Ensemble Accuracy
5 Cell Types
4,155 Training Images
2 AI Models

🤖 Technology Stack

  • ResNet152: Deep residual network with 152 layers for feature extraction
  • VGG16: Visual Geometry Group network with 16 layers
  • Ensemble Learning: Weighted averaging of multiple models for improved accuracy
  • TensorFlow & Keras: Industry-standard deep learning frameworks
  • Transfer Learning: Pre-trained on ImageNet for better generalization

🎯 How It Works

  1. Image Upload: You upload a cervical cell image through the web interface
  2. Preprocessing: The image is resized to 224×224 pixels and normalized
  3. Feature Extraction: Both ResNet152 and VGG16 extract relevant features from the image
  4. Ensemble Prediction: Predictions from both models are combined using weighted averaging
  5. Results Display: The system shows the predicted cell type with confidence scores for all categories

📈 Model Training Process

  • Dataset balanced using random oversampling to handle class imbalance
  • Data augmentation applied (rotation, shifts, flips, zoom) to improve generalization
  • Transfer learning from ImageNet pre-trained weights
  • Fine-tuning of top layers for cervical cell classification
  • Early stopping and learning rate reduction to prevent overfitting
  • Ensemble weights optimized based on validation performance
